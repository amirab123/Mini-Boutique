#  🛒 Mini-Boutique Vue.js + Tailwind



Projet simple en **Vue.js 3 (CDN)** et **Tailwind CSS**, réalisé en une seule page (SPA sans build tools).

---

## 🚀 Fonctionnalités

- 📦 Affichage de produits
- 🛒 Ajout au panier
- ❌ Suppression d’un produit du panier
- 💰 Calcul automatique du total
- 🎨 Interface responsive avec Tailwind

---

## 🧰 Technologies utilisées

- HTML5
- Vue.js 3 (CDN)
- Tailwind CSS (CDN)

---

## 📂 Structure du projet
index.html → application complète (Vue + UI)

## ▶️ Comment lancer le projet

1. Télécharger le fichier `index.html`
2. Ouvrir dans un navigateur (Chrome, Edge, Firefox)
3. Aucune installation requise
